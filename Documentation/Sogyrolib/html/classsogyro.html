Detailed Description

Class that interfaces the arduino connected to a gyro scope.

A class that interfaces the Slave Arduino Nano through a spi bus that is dedicated only to the Arduino Nano. The Arduino Nano is connected to a L3G4200 gyroscope. The functions of this class send register adresses to the Arduino Nano and then Arduino Nano returns the correpsonding data.
